Ce document présente une étude approfondie sur l’apprentissage passif des machines de Moore à partir de simples traces entrée–sortie, sans utiliser de requêtes d’équivalence ou d’appartenance. Les auteurs proposent trois algorithmes, analysent leurs propriétés théoriques et comparent leurs performances expérimentalement.
